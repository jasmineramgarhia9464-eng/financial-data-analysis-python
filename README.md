# financial-data-analysis-python
applied financial data analysis using python for economics and  finance research
## Financial Data Analysis Using Python

This repository demonstrates foundational financial data analysis techniques
used in economics and finance research, with a focus on reproducibility
and interpretation rather than black-box modelling.

### Research Question
How do financial asset returns behave over time, and what descriptive
patterns can be observed in market data?

### Data
Sample daily price data for a publicly traded asset.

### Methods
- Return calculation
- Descriptive statistics
- Visual exploration of financial time series

### Tools
Python (pandas, numpy, matplotlib)
